# `<my-element>`

## Usage

### npm install

```shell
npm i native-web-components-1
```

### CDN

```js
<script src="https://unpkg.com/native-web-components-1"></script>
```

### HTML tag

```html
<my-element initial-value="100"></my-element>
```

## Attributes

| Name            | Description                                    |
| --------------- | ---------------------------------------------- |
| `initial-value` | Supports a customizable initial counter value. |
